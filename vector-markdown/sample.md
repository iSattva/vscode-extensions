# Vector Markdown Sample

Open this file, then run **Vector Markdown: Open Branded Preview** to see the active theme applied.

## Why this matters

Enterprises want a **consistent, branded reading experience** for internal docs, and a **one-click way** to hand those docs to business stakeholders as PDF, HTML, or DOCX.

## A quick table

| Format | Engine                          | Extra install needed |
| ------ | -------------------------------- | --------------------- |
| HTML   | Built-in renderer                | No                     |
| PDF    | Local Chrome/Edge (chrome-launcher) | No, if a browser is installed |
| DOCX   | Pandoc, falling back to a JS converter | No (Pandoc optional, for best fidelity) |

> Switch themes any time with **Vector Markdown: Select Preview Theme**.

```js
console.log("Code blocks are themed too.");
```
